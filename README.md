# Robinson’s Curvival
Deck/Survival карточная игра на Unity.


## Требования
- Unity <укажите версию, например 2022.3 LTS>
- Git LFS
- Rider/VS 2022


## Быстрый старт
1. git clone ...
2. git lfs install
3. Откройте проект в Unity (версия ровно как в /ProjectSettings/ProjectVersion.txt)
